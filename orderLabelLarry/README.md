# Order Label Generator


### Description:

This script is designed to streamline the process of printing order label forms. Instead of editing a Google Doc, PDF, or using an external application, one concise window will provide you with everything that you need. It can be used to create order labels for boxes or pallets and will automatically send the sheet to the default printer.


### How to use:

Upon running the script, a window will pop up requesting the following information: 

- Order number 
- Organization name
- Contents of the box/pallet
- Box or a pallet
- Current and total box/pallet # (eg. Pallet 1 of 2)

Simply input this information and click 'Send.' The script will format the sheet and print it at the default printer location. Afterwards, a prompt will appear asking whether the user would like to repeat the process and complete another order form. Click 'Yes' to continue, or 'No' to exit the program. 


### Template and Adjustments:

This script utilizes a template document within a shared drive. Do not change any text containing an 'x' at the beginning. If you would like to edit the left column to change a row, you may do so, but this would change the template for all users (eg. 'Laptops + Chargers' -> 'Laptops'). The best practice is to change the template back to the original state after any edits. Talk to your supervisor to gain access to the shared drive.


### Additional Notes:

The auto-print feature of this script relies on Adobe Acrobat since some shop computers have certain compatibility issues using solely Windows print commands. Adobe Acrobat does not need to be opened for this script to work, as long as it is at least installed and has been opened one time (ever). The expected Adobe Destination is the default one: "C:\Program Files\Adobe\Acrobat DC\Acrobat\Acrobat.exe"
